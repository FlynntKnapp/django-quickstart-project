# Django Quick-Start Project

* Basic Django project with no extra apps.
* This project will display the Django green rocket at server root:
  * <http://localhost:8000/>
* Items included:
  * `pipenv` files for virtual environment:
    * `Pipefile`
    * `Pipefile.lock`
  * `django` for web framework.
